## TNT Countdown
Displays a countdown timer above primed TNT. (Original idea by Sk1er)

---

### Changes from Original

- Uses vanilla rendering (for mod compatibility)
- Adds customization with OneConfig.
  - Custom Text
  - Custom Text color
- No longer depends on essential.
- Supports more servers (Join the [Discord](https://discord.gg/gGuQ55PCRn) to suggest other servers!)
    - Minemen

### Planned features...

- Custom fuse times
  - Allow user defined (or create a database?) custom TNT fuse times for servers that modify it.
